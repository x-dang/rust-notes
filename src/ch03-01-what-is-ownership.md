## 什么是所有权

所有程序都需要在运行时它们管理使用的内存. 有些语言提供了垃圾回收功能, 并不时地检查不再使用的内存;
有些语言需要程序员手动地申请和释放内存. Rust 则使用第三种方式: 通过所有权系统管理内存,
其通过一系列的规则, 在编译时检查内存的使用情况. 所有权系统并不会减慢程序的运行速度.


### 所有权规则

* 每个值都有一个被称为 *所有者*(owner) 的变量.
* 每个值同时只能有一个所有者.
* 当所有者离开作用域, 该值将被丢弃.


### 变量作用域

变量作用域就是程序中变量有效的范围.

```rust
{                      // s 在这里无效, 它还没有被定义
    let s = "hello";   // s 从这里开始有效

    // do stuff with s
}                      // 该作用域在这里已结束, s 不再有效
```
